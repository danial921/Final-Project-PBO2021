# Final-Project-PBO2021

**Deskripsi Topik**
```
Topik FP yang kami ambil adalah game sederhana.
Game yang kami ambil adalah Snake Game, yaitu permainan dengan tujuan menggerakan ular 
untuk memakan setiap bola / objek yang ditampilkan di layar sebanyak=banyaknya dengan 
peraturan tidak boleh menabrak dinding ataupun menabrak diri ular itu sendiri. 

Kami mengimplementasikan program Snake Game yang sudah ada di internet 
yang kemudian kami tambahkan beberpa fitur yang belum ada dalam program tersebut.

Fitur-fitur yang kami tambahkan adalah sebagai berikut.
1. Fitur Title Screen
2. Fitur Level difficulty (Easy, Medium, Hard)
3. Fitur HighScore
```
